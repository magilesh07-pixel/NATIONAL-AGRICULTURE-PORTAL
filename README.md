🌾 Vivasayi — National Agriculture Portal
Vivasayi is an official AI-powered farming support portal designed to empower Indian farmers with digital intelligence. Built as part of the National Agriculture Digital Mission, it provides critical information, financial tracking, and diagnostic tools in a localized, user-friendly interface.

🚀 Key Features
🤖 AI Farming Assistant: Multilingual support (English, Hindi, Tamil, Telugu, etc.) for crop planting, fertilizer strategies, and pest management.
📊 Institutional Ledger: A persistent expense tracker with SQLite and LocalStorage fallbacks, ensuring financial data is never lost across sessions.
🏥 Farm Doctor: Computer Vision node for high-fidelity crop imagery analysis and pathogen identification.
☀️ Agromet Advisory: Real-time localized weather monitoring and harvesting condition alerts (Precision Geolocation enabled).
🇮🇳 Multilingual Node: Native language support for the entire Indian agricultural community.
🛠️ Technology Stack
Backend: Python (Flask)
Database: SQLite (Relational Persistence)
Frontend: Vanilla JS (ES6+), CSS3 (Modern Glassmorphism Design)
Intelligence: PIL (Image Processing Algorithm), Custom Knowledge Base Logic
